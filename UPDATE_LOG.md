# WebDisplays 1.21.1 更新日志

发布日期：2026-04-29

## 新增

- 新增网页内虚拟导航按钮（左上角）：
  - `=` 展开/收起
  - `<` 后退
  - `>` 前进
  - `R` 刷新
- 新增同标签跳转兼容逻辑，处理 `_blank` / `window.open` 场景，减少站点点击无响应问题。
- 新增屏幕 URL 客户端到服务端同步包，支持屏幕内导航后同步给其他玩家。

## 修复

- 修复屏幕方块在特定视角（右侧斜视）出现黑面/不渲染问题（渲染包围盒与剔除状态修正）。
- 修复屏幕贴图异常（错误显示为图集拼贴）问题，校正 UV 与渲染参数。
- 修复声音不会随距离衰减问题，恢复按距离动态音量更新。
- 修复第一人称手持渲染崩溃（`ClientProxy.onRenderPlayerHand` 空指针）问题。
- 修复部分网页（如 B 站）点击后不跳转的问题（多层兼容处理）。

## 交互增强

- 键盘界面与 MinePad 增加浏览器快捷键：
  - `Alt + ←` 后退
  - `Alt + →` 前进
  - `F5` / `Ctrl + R` 刷新
  - `Ctrl + Shift + R` 强制刷新（忽略缓存）

## 服务器使用说明

- 目前为“本地浏览器渲染 + 状态同步”方案：
  - 每位玩家本地打开网页并渲染；
  - URL 与交互状态在服务器中同步；
  - Cookie 仍保存在玩家本地环境，不在服务器统一托管。

## 已知限制

- 不同网站前端策略差异较大，少数站点仍可能存在个别点击路径不触发跳转。
- 跨客户端登录状态（Cookie 完全共享）不在本次更新范围内。
